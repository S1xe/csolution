较难知识点

-   a->b 表示的是(*a).b 其中 a 为结构体指针,*为解引用 .为获取成员变量
-   a[b] 表示的是\*(a+b) 假定 a 为变量名,b 为下标 表示取从 a 起往后第 b 个(首位为 0)元素的值

    > 如 int a[10]={1,2,3,4,5,6};
    > a[0] 取到的是 1
    > a[2]取到的是 3
